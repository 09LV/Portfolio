Infinite Runner

×

Project Overview

An addictive endless runner with automatic forward movement, responsive jump mechanics, and procedurally generated obstacles. Built to showcase clean gameplay programming and smooth character controls.

Key Features

  • Character Movement: Smooth acceleration with responsive jump mechanics (including coyote time and jump buffering)
  • Obstacle Generation: Infinite procedural level generation with difficulty progression
  • Score System: Coin collection with combo multipliers and visual feedback
  • Visual Effects: Particle systems for coin collection, screen shake on collisions

Development Notes

  • Implemented object pooling for obstacles and coins to optimize performance
  • Created a dynamic difficulty system that increases speed and obstacle density over time
  • Solved ground detection issues with a custom raycast-based solution
  • Developed a smooth camera follow system using Cinemachine with collision avoidance
  • Added magnet power-up that attracts nearby coins using physics overlap spheres

Zombie Apocalypse

×

Project Overview

First-person shooter with advanced zombie AI behaviors and weapon systems. Designed to create intense combat scenariosm with increasing difficulty. Features multiple enemy types, health management, and wave-based progression.

Key Features

  • AI System: Finite State Machine controlling idle, chase, attack, and death states
  • Weapon Mechanics: Raycast-based shooting with recoil, reloading, and ammo management
  • Wave System: Progressive difficulty with increasing enemy counts and types
  • Health System: Player health with visual feedback and zombie attack damage
  • Navmesh Navigation: Efficient pathfinding for zombie movement

Development Notes

  • Implemented object pooling for zombie instantiation to optimize performance
  • Created a custom animation controller for zombie behaviors
  • Developed a dynamic spawn system that adjusts based on player position
  • Added headshot detection with bonus damage
  • Implemented screen effects for player damage feedback
  • Optimized AI logic to maintain performance with many active enemies
